LLM Council — AI 모델 회의 시뮬레이터

LLM Council은 Andrej Karpathy가 공개한 오픈소스 프로젝트로, 여러 개의 LLM이 동시에 토론하고 서로를 평가해 최종 결론을 도출하는 “AI 전문가 위원회(Expert Council)” 시스템으로 출시 하루 만에 GitHub 2,000개를 돌파하며 큰 관심을 끌고 있습니다..

1. 핵심 개념

  • GPT, Claude, Gemini 등 다양한 AI 모델을 동시에 실행
  • 각 모델이 답변을 내고, 다른 모델의 답변을 리뷰
  • ‘의장(Chairman)’ 역할을 맡은 모델이 결과를 종합하여 최종 결론 생성
  • 완전 오픈소스로 누구나 수정 및 확장 가능

2. 동작 방식

  • ① 1차 답변 생성 — 여러 모델이 동시에 답을 생성
  • ② 상호 평가 — 모델끼리 서로의 답변을 익명으로 평가/순위화
  • ③ 최종 결론 — 의장 모델이 모든 평가 결과를 통합해 최종 답변 작성

3. 기술적 특징

  • FastAPI 기반 백엔드
  • React + Vite 프런트엔드
  • OpenRouter API로 여러 AI 모델 연결
  • 대화 데이터는 로컬 JSON 파일로 저장

4. 활용 분야

  • 정책/전략 검토
  • 복잡한 문제 해결
  • 코딩 오류 분석
  • AI 교육용 시뮬레이션
